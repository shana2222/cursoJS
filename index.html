Funcionamiento del programa

Este programa permite realizar pedidos de hamburguesas personalizadas. El proceso es el siguiente:

  1. Debes dirigirte al menú PEDIDOS
  2. El usuario ingresa la cantidad de combos que desea pedir.
  3. Para cada combo, se solicita la siguiente información:
    • Nombre del cliente.
    • Tipo de proteína para la hamburguesa (con un costo de $250).
    • Cantidad de carnes deseadas en la hamburguesa (cada carne tiene un extra de $50).
    • Condimentos deseados.
    • Si se desea agregar papas fritas (con un costo adicional de $75).
  4. Una vez que se han ingresado los datos, se muestra el ticket del pedido en la consola y en el HTML.
  5. Se repite el proceso para cada combo solicitado.
